# Installation

## Kali Linux and Parrot Security

Step 1. git clone

Step 2. cd Darkmap/

Step 3. pip install python-nmap

Step 4. python3 darkmap.py --help

## Black Arch and Arch Linux

### COMING SOON

## Termux

Step 1. git clone

Step 2. cd Darkmap/

Step 3. bash install.sh

Step 4. python3 darkmap.py --help
